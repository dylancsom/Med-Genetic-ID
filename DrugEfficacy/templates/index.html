DNA Reader

How It Works

Once users upload a text file with their DNA data (provided through services such as Ancestry or 23andme) this algorithm reads the file line by line and identifies the RSIDs and the Alleles attached to them. Together, these form an SNP. Some SNPs have been studied and found to hold some association to cancer in those studies. A number of these SNPs have been saved to an accompanying database, where the identified SNPs in user's DNA text file are compared. If there are any matches found, meaning if there is an SNP found in both the user's DNA and the database, it will in the results, along with the type of cancer, and the risk association the SNP holds.
